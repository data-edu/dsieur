Data Science in Education Using R


Ryan A. Estrellado, Emily A. Freer, Jesse Mostipak, Joshua M. Rosenberg, and Isabella C. Velásquez

Welcome to the supplementary website for Data Science in Education Using R.

This website provides additional content and resources to supplement the hard copy such as:

  • A hyperlink to the {dataedu} package website.
  • External blog posts that we have written on the book.

Contributing to the Book

We thank all of you who have provided guidance, feedback, and fixes for the book.

Here are various ways to contribute further:

  • Submit an “issue” to our GitHub repository that describes a data science problem that is unique to the education setting
  • Submit a pull request to share a solution for the problems discussed in the book to the education setting
  • Share an anonymized dataset for use in the book (or a future version of it)

If you are new to data science in education, welcome! We would love to have your feedback by email.

We hope that as the book evolves, it grows to reflect the changing needs of data scientists in education.
